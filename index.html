📐 5 ракурсів (на SKU)

  • 4 основні — товар з усіх сторін, строго 90° без нахилу (фронт / правий бік / тил / лівий бік). Пляшка й банка = об'ємний товар → 4 сторони.
  • 1 маркетинговий — під кутом зверху 30°–60° (для пляшок), «найапетитніший», з акуратними відблисками.

💧 1 мокап із краплями

Фронтальний кадр із конденсатом/краплями на тарі — дозволено лише для CSD (газовані напої), Pepsi підходить. Краплі натуральні, не перекривають лого й текст етикетки.

Візуальний еталон «капельок» — приклад-референс від замовника (надане фото 0,5 л). Можу вбудувати його в бриф, щойно покладеш файл у репозиторій.

Опційно для мереж AFH / METRO: додатково основні ракурси під нахилом зверху 20°–30° для товарів до 0,5 л або до 25 см заввишки (стосується банок 0,33 л і пляшок 0,5 л). Уточнити в замовника, чи входить у цей пакет.

3Активи (що передано)

🧴 3D-модель

Базова банка — Beverage Can (FBX, текстури вкладені + окремо).
Геометрія: тіло банки + кришка; матеріали Steel / can_top / Label.

Етикетки на моделі — дефолтні. Підмінити на відповідну UKR0xx-текстуру в слот Label.

5Технічні вимоги до виводу

Обов'язкові параметри зі стандарту Listex + PepsiCo. Виробничі рекомендації — підсвічено, їх можна коригувати, не порушуючи обов'язкових.

Обов'язкові (Listex / PepsiCo)

Формат
PNG, без фону (прозорий).
Роздільність
не менше 2304×2304 px.
DPI
не менше 72 dpi.
Колір
RGB, натуральні природні кольори.
Світлотінь
Натуральний світлотіньовий малюнок — без засвітів і без провалів у тінях.
Ретуш
Без дефектів, бруду й пилу; без зайвих елементів. Вирівняні форма й перспектива. Без ліній фальцювання / складання / біговки.
Кути
Основні — строго 90° без нахилу; маркетинговий — зверху 30°–60°.
Текст
На фото обов'язкова інформація українською. Якщо її нема на тарі — окремим кадром етикетка з перекладом.
Загальне
Товар має виглядати естетично й натурально — так, як він продається в мережі.

Виробничі рекомендації (для рендеру)

Рушій
Cycles (фотореалізм, GPU). EEVEE — лише для чернеток.
Камера
Довгий фокус (85–135 мм) — мінімум перспективних спотворень; центр на етикетці.
Освітлення
Студійне: софт-бокс ключ + контровий рим. Рівні відблиски на склі/металі, без пересвітів етикетки.
Матеріали
Пляшка — прозорий PET (Transmission), банка — алюміній (метал). Етикетка — у слот Label, без зайвого roughness.
UV / посадка етикетки
Текстура 8080×2080 лягає по обводу тари; перевірити шов ззаду й вертикальні пропорції тексту.
